# Dann Kang'ere Portfolio

This is a personal portfolio website for **Dann Kang'ere**, an AERD student at JKUAT with a passion for technology, farming innovation, and creative projects.  
The site showcases his biography, CV, hobbies, projects, and contact information.

---

## 🌐 Website Structure
- **Home**: Introduction and tagline
- **About**: Biography, hobbies, music & movies, and CV details
- **Projects**: Highlighted projects (e.g., chicken farming project)
- **Contact**: Email, phone, and address

---

## 🚀 Deployment
The site is hosted using **GitHub Pages**.  
To deploy:
1. Fork or clone this repository.
2. Push changes to the `main` branch.
3. In GitHub settings, enable Pages for the repository.
4. The site will be available at `https://<username>.github.io/`.

---

## 🛠️ Technologies Used
- **HTML5** for structure
- **CSS3** for styling
- **GitHub Pages** for hosting

---

## 📌 Customization
- Update `about.html` with new biography or CV details.
- Add new projects in `projects.html`.
- Replace images in the `images/` folder.
- Adjust colors and fonts in `style.css`.
